This is the slider plugin which is very useful to slide images with full functionality such as arrows, customize image width and height.

== Description ==
Chintan Slider plugin has functionality to choose images from your local computer as well as you can choose from media library, this plugin provides beautiful animations also user can change images, edit images and delete too.


Ordered list:

1. add, edit and delete image
2. adjust width and height
3. user can add this slider anywhere in the website just they have to add shortcode on particular page



== Installation ==
1. Upload "chintan-slider-plugin.php" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Place "do_action( 'chintan_slider_plugin_hook' );" in your templates.
